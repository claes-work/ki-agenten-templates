# KI-Agenten Templates (n8n)

Importierbare [n8n](https://n8n.io)-Workflow-Templates aus meinen YouTube-Tutorials auf
[@claes-ki](https://www.youtube.com/@claes-ki) — herunterladen, importieren, eigene
Zugangsdaten hinterlegen, fertig.

## Templates

| Datei | Workflow |
|---|---|
| [content-writer-agent.json](content-writer-agent.json) | Multi-AI-Agents Content-Writer — mehrere KI-Agenten schreiben gemeinsam Content (12 Nodes) |
| [eür-ki-agent-template.json](eür-ki-agent-template.json) | Invoice-/EÜR-Agent — Rechnungen automatisch erfassen und verbuchen (17 Nodes) |

## Import in n8n

1. Template als `.json` herunterladen
2. In n8n: **Workflow → Import from File…**
3. Credentials (z. B. OpenAI, Gmail) in den Nodes neu verbinden
